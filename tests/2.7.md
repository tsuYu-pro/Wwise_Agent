我开关think功能后用聊天对话询问是否开启Think机制，以下是他的回答：

` 标签是工具系统注入的提示语**，并非 Anthropic 官方的 Extended Thinking 功能。

具体来说：
- 每次工具调用返回结果后，系统会在结果末尾附加提示：`"你的下一条回复必须以 <think> 标签开头"`
- 这是 **Wwise Agent 的 Prompt 工程设计**，用于引导我分析工具执行结果
- 并非 Anthropic API 层面的 `extended_thinking` 参数开启

**结论：官方 Extended Thinking 功能未开启**，`<think>` 只是本应用的提示词机制。
